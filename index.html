uma linguagem usada no desenvolvimento de uma nova versão
do sistema operacional Unix, mas hoje é aplicada para
criar softwares.		 A linguagem C tomou existência nos anos de 1969 - 1973 É muito usada em banco de dados para todos os tipos
de sistemas: financeiro, governamental, mídia, entretenimento,
telecomunicações, saúde, educação, varejo, redes sociais, etc.
Grandes empresas como Apple, Microsoft, Oracle usam a linguagem C.		 Visto que ela foi criada na década de 70 com o intuito
